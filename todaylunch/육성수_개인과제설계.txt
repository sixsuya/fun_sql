1. 프로젝트명 : 오점뭐(오늘 점심 뭐먹지?)

2. 유형 : 회원관리(단, 회원이 사람이 아니라 식당)

3. 개요 

 - 학원 주변에 점심식사 가능한 식당을 리스트업 해서 관리
 - '등록' 기능을 통해 식당 등록
   
  * '등록' label
    1) 식당명
    2) 주소
    3) 요식업 구분(프렌차이즈, 중식, 일식, 한식 등)
    4) 대표메뉴
    5) 식사가격
    6) 최소 소요 인원
    7) 구글 or 네이버 평점
    8) 할인 또는 쿠폰 적용가
    9) 비고

 - '등록' 탭을 통해 업로드한 식당은 '목록' 탭을 통해 리스트업 진행

  * '목록' label 요약 표기
  1) 식당명
  2) 요식업 구분
  3) 식사가격
  4) 할인 또는 쿠폰 적용가
  5) 최소 소요 인원

  =>  가능하면 sorting function 구현
  => 클릭하면 상세 내용이 나오도록 구현
     '상세내용' 에서는 '등록' 에서 입력한 row값이 모두 나오도록 구현
     - 추가로 '방문완료' submit button을 구현
     - submit click 시,  POPUP으로 별점 및 후기 입력 기능 구현
      =>  submit 완료한 식당은 '목록'에서  '방문완료' 탭으로 탭 이동


 '방문완료' 탭
 - 이전 별점 및 후기를 기존 row data와 같이 볼 수 있도록 정리
